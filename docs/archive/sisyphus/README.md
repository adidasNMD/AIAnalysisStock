# Sisyphus Archive

This directory preserves Markdown notes that were previously kept under `.sisyphus/`.

Retention policy:
- `plans/`: promoted here because they contain repo-level implementation context.
- `notepads/`: archived for traceability, but treated as historical working notes rather than source-of-truth docs.

Operational rule:
- `.sisyphus/` is now considered local scratch space and is ignored by Git.
- New durable plans and architecture decisions should go under `docs/`, not `.sisyphus/`.
